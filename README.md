# Enterprise Storage RMA Engine

## Overview
A production-inspired RMA (Return Merchandise Authorization) system 
designed for enterprise hardware manufacturers.

## Architecture
- PostgreSQL
- FastAPI (planned)
- Dockerized deployment
- Event-driven audit logging

## Features
- Policy-based eligibility engine
- Refund & replacement workflow
- Shipment lifecycle tracking
- Event logging
